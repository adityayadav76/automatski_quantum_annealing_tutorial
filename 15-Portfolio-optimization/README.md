# Portfolio Optimization (QUBO)

## Overview
Portfolio optimization aims to allocate investments across assets to maximize return while minimizing risk.

## QUBO Formulation
- Binary variables for selecting assets
- One-hot constraint for budget allocation
- Minimize risk (variance) while maximizing return
