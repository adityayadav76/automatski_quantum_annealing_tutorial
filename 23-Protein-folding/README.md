# Protein Folding (QUBO)

## Overview
Protein folding predicts the 3D structure of proteins. We use a simplified lattice model formulated as QUBO.

## QUBO Formulation
- Binary variables for amino acid positions
- Minimize energy (hydrophobic contacts)
- Chain connectivity constraints

## Running the Example
```bash
python step23_protein_folding.py
```
