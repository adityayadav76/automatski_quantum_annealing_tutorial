# Automatski Quantum Annealing Tutorial
The Quantum Annealing Tutorial Using Automatski's Quantum Annealers

![](https://automatski.com/wp-content/uploads/2025/05/Automatski-New-Logo.svg)

## About

This quantum annealing tutorial was created by [Automatski](https://automatski.com). It is preconfigured to run with Automatski's Cloud Based Entry Level Annealers. So you can get started rightaway. There is no need to signup, credit cards, registration, purchasing subscriptions. It works out of the box and is perhaps the easiest way to start with quantum annealing.

Automatski's quantum annealers support upto 500 million spin up-down boolean variables.

## Installation

Requires Python v3.11+ to run.
Install dependencies:

```sh
pip install -r requirements.txt
```
