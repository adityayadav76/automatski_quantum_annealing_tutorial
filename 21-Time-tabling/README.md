# Time Tabling / Scheduling (QUBO)

## Overview
University timetabling assigns courses to time slots and rooms while avoiding conflicts.

## QUBO Formulation
- Binary variables for course-time-room assignments
- One-hot constraints for courses
- Conflict avoidance constraints

## Running the Example
```bash
python step21_time_tabling.py
```
