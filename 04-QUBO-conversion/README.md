# QUBO Conversion

## Overview
This step demonstrates how to convert various problem formulations into QUBO format and how to extract the QUBO matrix from a BQM.

## Key Operations
- `bqm.to_qubo()`: Convert BQM to QUBO format
- `bqm.to_ising()`: Convert BQM to Ising format
- Converting from other problem types to QUBO

## Running the Example
```bash
python step4_qubo_conversion.py
```
